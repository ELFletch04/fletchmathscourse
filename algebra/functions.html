Introduction to Functions

In mathematics, a function is a relation between a set of inputs and a set of permissible outputs. Each input is related to exactly one output.

Basic Concepts

Function Notation

Functions are typically denoted by letters such as \( f \), \( g \), or \( h \). The notation \( f(x) \) represents a function named \( f \) with \( x \) as the input variable.

\[ f(x) = 2x + 3 \]

In this example, \( f(x) \) is a function where \( 2x + 3 \) represents the output for any input \( x \).

Example:

For \( f(x) = 2x + 3 \), find \( f(4) \):

\[ f(4) = 2(4) + 3 = 8 + 3 = 11 \]

Therefore, \( f(4) = 11 \).

Domain and Range

The domain of a function is the set of all possible inputs (x-values) for which the function is defined. The range is the set of all possible outputs (y-values).

Example:

For \( f(x) = \sqrt{x} \), the domain is \( x \geq 0 \) and the range is \( y \geq 0 \).

For \( g(x) = \frac{1}{x} \), the domain is \( x \neq 0 \) and the range is \( y \neq 0 \).

Composite Functions

A composite function is formed when one function is applied to the result of another function. If \( f \) and \( g \) are functions, the composite function \( f(g(x)) \) is defined as:

\[ (f \circ g)(x) = f(g(x)) \]

Example:

If \( f(x) = 2x + 1 \) and \( g(x) = x^2 \), then \( (f \circ g)(x) = f(g(x)) = f(x^2) = 2x^2 + 1 \).

Find \( (g \circ f)(x) \):

\[ (g \circ f)(x) = g(f(x)) = g(2x + 1) = (2x + 1)^2 = 4x^2 + 4x + 1 \]

Therefore, \( (g \circ f)(x) = 4x^2 + 4x + 1 \).

Inverse Functions

An inverse function reverses the operation of the original function. If \( f \) is a function, its inverse is denoted by \( f^{-1} \). For \( f \) and \( f^{-1} \):

\[ f(f^{-1}(x)) = x \text{ and } f^{-1}(f(x)) = x \]

Example:

If \( f(x) = 3x - 4 \), then to find \( f^{-1}(x) \), solve \( y = 3x - 4 \) for \( x \):

\[ y = 3x - 4 \]

Solve for \( x \):

\[ x = \frac{y + 4}{3} \]

Therefore, the inverse function is:

\[ f^{-1}(x) = \frac{x + 4}{3} \]

Transformations of Functions

Transformations change the position or size of the graph of a function. Common transformations include translations, reflections, stretches, and compressions.

  • Translations: \( f(x) + c \) (vertical) or \( f(x + c) \) (horizontal)
  • Reflections: \( -f(x) \) (in x-axis) or \( f(-x) \) (in y-axis)
  • Stretches: \( af(x) \) (vertical by factor \( a \)) or \( f(bx) \) (horizontal by factor \( \frac{1}{b} \))

Example:

Given \( f(x) = x^2 \):

  • Translation: \( f(x) + 3 \) shifts the graph up by 3 units.
  • Translation: \( f(x - 2) \) shifts the graph right by 2 units.
  • Reflection: \( -f(x) = -x^2 \) reflects the graph in the x-axis.
  • Reflection: \( f(-x) = (-x)^2 = x^2 \) reflects the graph in the y-axis (no change for this specific function).
  • Stretch: \( 2f(x) = 2x^2 \) stretches the graph vertically by a factor of 2.
  • Stretch: \( f(2x) = (2x)^2 = 4x^2 \) compresses the graph horizontally by a factor of \(\frac{1}{2}\).

Examples and Applications

Let's look at some specific examples and how to solve them:

1. Find \( (f \circ g)(x) \) where \( f(x) = 2x + 3 \) and \( g(x) = x^2 \):

\[ (f \circ g)(x) = f(g(x)) = f(x^2) = 2x^2 + 3 \]

2. Find the inverse of \( f(x) = 2x + 3 \):

\[ y = 2x + 3 \]

Solve for \( x \):

\[ x = \frac{y - 3}{2} \]

Therefore, the inverse function is:

\[ f^{-1}(x) = \frac{x - 3}{2} \]

3. Apply a transformation to \( f(x) = x^2 \) for a vertical stretch by 3 and a horizontal translation by 2:

\[ g(x) = 3(x - 2)^2 \]

Practice Questions

Solve the following:

1. If \( f(x) = x^2 + 1 \) and \( g(x) = 3x - 2 \), find \( (f \circ g)(x) \).

2. Determine the inverse of \( h(x) = \frac{x + 1}{x - 1} \).

3. Apply the transformation \( f(x) = 2x + 3 \) for a reflection in the y-axis.
