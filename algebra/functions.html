Introduction to Functions

In mathematics, a function is a relation between a set of inputs and a set of permissible outputs. Each input is related to exactly one output.

Basic Concepts

Function Notation

Functions are typically denoted by letters such as \( f \), \( g \), or \( h \). The notation \( f(x) \) represents a function named \( f \) with \( x \) as the input variable.

\[ f(x) = 2x + 3 \]

In this example, \( f(x) \) is a function where \( 2x + 3 \) represents the output for any input \( x \).

Example:

For \( f(x) = 2x + 3 \), find \( f(4) \):

\[ f(4) = 2(4) + 3 = 8 + 3 = 11 \]

Therefore, \( f(4) = 11 \).

Try this question:

Consider the function:

\[ f(x) = 2x + 3 \]

What is the value of \( f(4) \)?

A. \( 8 \)
B. \( 9 \)
C. \( 10 \)
D. \( 11 \)

Solution:

  1. Substitute \( x = 4 \) into the function \( f(x) = 2x + 3 \):
  2. Calculate \( f(4) = 2(4) + 3 = 8 + 3 = 11 \).
  3. Therefore, \( f(4) = 11 \).

Domain and Range

The domain of a function is the set of all possible inputs (x-values) for which the function is defined. In other words, it’s the collection of x-values that you can put into the function without causing any undefined mathematical operations, such as dividing by zero or taking the square root of a negative number.

The range of a function is the set of all possible outputs (y-values). These are the values that come out of the function when you substitute x-values from the domain. The range can depend on the type of function and any restrictions placed by the operations within it.

Understanding Domain

To find the domain of a function, you need to identify any restrictions on the input values (x). Common restrictions include:

  • Division by zero: In functions where x appears in the denominator, such as \( g(x) = \frac{1}{x} \), the function is undefined at \( x = 0 \) because division by zero is not allowed.
  • Square roots of negative numbers: In functions involving square roots, such as \( f(x) = \sqrt{x} \), the function is only defined when \( x \geq 0 \), since the square root of a negative number is not a real number.

Understanding Range

To find the range of a function, you need to determine the set of y-values that result from plugging in all the x-values from the domain. Some key things to consider:

  • Constant functions: A function like \( f(x) = 3 \) has a domain of all real numbers (since no restrictions exist on x), but the range is just \( y = 3 \), because no matter the input, the output is always 3.
  • Quadratic functions: For a quadratic function like \( f(x) = x^2 \), the range is \( y \geq 0 \), because squaring any real number results in a non-negative output. The domain is all real numbers.
  • Linear functions: A linear function such as \( f(x) = 2x + 1 \) has a domain and range of all real numbers because there are no restrictions on x or y.

Examples of Domain and Range

Let’s look at some specific examples:

  • Example 1: For \( f(x) = \sqrt{x} \), the domain is \( x \geq 0 \), because square roots of negative numbers are undefined. The range is \( y \geq 0 \), since the square root always produces non-negative results.
  • Example 2: For \( g(x) = \frac{1}{x} \), the domain is \( x \neq 0 \), because the function is undefined at \( x = 0 \). The range is also \( y \neq 0 \), because the reciprocal of any non-zero number is never zero.
  • Example 3: For \( h(x) = x^2 \), the domain is all real numbers (\( x \in \mathbb{R} \)), but the range is \( y \geq 0 \), since squaring any number gives a non-negative result.

Writing Domain and Range in Set Notation

In set notation, the domain and range can be written to clearly specify the sets of values. Here’s how to express them:

  • Domain: The domain can be expressed as a set of x-values. For example, the domain of \( f(x) = \sqrt{x} \) is written as: \[ \{ x : x \geq 0 \} \]. This means "the set of all x such that x is greater than or equal to 0."
  • Range: Similarly, the range can be expressed as a set of y-values. For example, the range of \( f(x) = \sqrt{x} \) is written as: \[ \{ y : y \geq 0 \} \]. This means "the set of all y such that y is greater than or equal to 0."
  • Example: For the function \( g(x) = \frac{1}{x} \), the domain is written as: \[ \{ x : x \neq 0 \} \], and the range is written as: \[ \{ y : y \neq 0 \} \].

Summary

When determining the domain and range of a function, consider the mathematical operations within the function and any restrictions they impose. The domain tells you which x-values can be input into the function, and the range tells you the possible y-values that result from those inputs. Writing these sets in set notation provides a clear and precise way to describe the valid input and output values.

Consider the function:

\[ f(x) = \frac{2x + 1}{x - 3} \]

Find the domain and range of the function and express them in set notation.

A. Domain: \(\{ x : x \neq 3 \}\); Range: \(\{ y : y \neq 0 \}\)
B. Domain: \(\{ x : x \geq 3 \}\); Range: \(\{ y : y \geq 0 \}\)
C. Domain: \(\{ x : x \leq 3 \}\); Range: \(\{ y : y \geq 0 \}\)
D. Domain: \(\{ x : x \neq 3 \}\); Range: \(\{ y : y \neq 0 \}\)

Solution:

  1. **Domain:** The function \( f(x) = \frac{2x + 1}{x - 3} \) is undefined when the denominator is zero. So, \( x \neq 3 \). Thus, the domain is: \[ \{ x : x \neq 3 \} \]
  2. **Range:** To find the range, observe that the function can approach all real values except for the value where the function’s output is zero, which happens when \( 2x + 1 = 0 \) or \( x = -\frac{1}{2} \). However, due to the asymptotic behavior, \( y \neq 0 \). Thus, the range is: \[ \{ y : y \neq 0 \} \]
  3. Therefore, the correct answer is A.

Composite Functions

A composite function is formed when one function is applied to the result of another function. If \( f \) and \( g \) are functions, the composite function \( f(g(x)) \) is defined as:

\[ f(g(x)) \]

Example:

If \( f(x) = 2x + 1 \) and \( g(x) = x^2 \), then \( f(g(x)) = f(x^2) = 2x^2 + 1 \).

To find \( g(f(x)) \), substitute \( f(x) \) into \( g(x) \). Wherever there is an \( x \) in \( g(x) \), replace it with \( f(x) \). Similarly, to find \( f(g(x)) \), substitute \( g(x) \) into \( f(x) \). Wherever there is an \( x \) in \( f(x) \), replace it with \( g(x) \). For example:

\[ g(f(x)) = g(2x + 1) = (2x + 1)^2 = 4x^2 + 4x + 1 \]

Therefore, \( g(f(x)) = 4x^2 + 4x + 1 \).

Consider the functions:

\[ f(x) = x + 3 \]

\[ g(x) = 2x - 1 \]

Find \( f(g(x)) \) and \( g(f(x)) \), and write the expressions in simplified form.

A.

\( f(g(x)) = 2x + 2 \);

\( g(f(x)) = 2x + 5 \)

B.

\( f(g(x)) = 2x + 5 \);

\( g(f(x)) = 2x + 5 \)

C.

\( f(g(x)) = 2x + 8 \);

\( g(f(x)) = 2x + 5 \)

D.

\( f(g(x)) = 2x + 5 \);

\( g(f(x)) = 2x + 7 \)

Solution:

  1. **Finding \( f(g(x)) \):** \[ f(g(x)) = f(2x - 1) \] Substitute \( g(x) \) into \( f(x) \): \[ f(2x - 1) = (2x - 1) + 3 = 2x + 2 \] So, \( f(g(x)) = 2x + 2 \).
  2. **Finding \( g(f(x)) \):** \[ g(f(x)) = g(x + 3) \] Substitute \( f(x) \) into \( g(x) \): \[ g(x + 3) = 2(x + 3) - 1 = 2x + 6 - 1 = 2x + 5 \] So, \( g(f(x)) = 2x + 5 \).
  3. Therefore, the correct answer is A.

Inverse Functions

An inverse function reverses the operation of the original function. If \( f \) is a function, its inverse is denoted by \( f^{-1} \). For \( f \) and \( f^{-1} \):

\[ f(f^{-1}(x)) = x \text{ and } f^{-1}(f(x)) = x \]

To find the inverse of a function, you can follow these steps:

  1. Replace \( f(x) \) with \( y \). This makes it easier to manipulate the equation. For example, if \( f(x) = 3x - 4 \), replace it with \( y = 3x - 4 \).
  2. Swap \( x \) and \( y \). This step helps to find the inverse function. So, the equation becomes \( x = 3y - 4 \).
  3. Make \( y \) the subject of the equation. Solve for \( y \) in terms of \( x \):
    4. \[ y = \frac{x + 4}{3} \]
  4. Change \( y \) back to \( f^{-1}(x) \). Therefore, the inverse function is:
    5. \[ f^{-1}(x) = \frac{x + 4}{3} \]

Consider the function:

\[ f(x) = 2x + 5 \]

Find the inverse function \( f^{-1}(x) \) and write the expression in simplified form.

A. \( f^{-1}(x) = \frac{x - 2}{5} \)
B. \( f^{-1}(x) = \frac{x + 5}{2} \)
C. \( f^{-1}(x) = \frac{x - 5}{2} \)
D. \( f^{-1}(x) = \frac{x - 5}{2} \)

Solution:

  1. Replace \( f(x) \) with \( y \):
    2. \[ y = 2x + 5 \]
  2. Swap \( x \) and \( y \):
    3. \[ x = 2y + 5 \]
  3. Make \( y \) the subject:
    4. \[ y = \frac{x - 5}{2} \]
  4. Thus, the inverse function is:
    5. \[ f^{-1}(x) = \frac{x - 5}{2} \]
  5. Therefore, the correct answer is C.

Transformations of Functions

Transformations change the position or size of the graph of a function. Common transformations include translations, reflections, stretches, and compressions.

  • Translations: These move the graph without changing its shape. There are two types of translations:
    • Vertical translation: This occurs when a constant \( c \) is added to or subtracted from the entire function, i.e., \( f(x) + c \). This shifts the graph up or down along the y-axis. A positive \( +c \) moves the graph upward by \( c \) units, while a negative \( -c \) moves it downward by \( c \) units.
    • Horizontal translation: This happens when \( c \) is added to or subtracted from the input \( x \) inside the function, i.e., \( f(x + c) \). This shifts the graph left or right along the x-axis. A positive \( +c \) actually moves the graph left by \( c \) units, while a negative \( -c \) moves it right by \( c \) units. This is the opposite of what you might expect because the change occurs inside the function.
  • Reflections:
    • \( -f(x) \) reflects the graph across the x-axis, flipping it upside down.
    • \( f(-x) \) reflects the graph across the y-axis, flipping it left to right.
  • Stretches and Compressions: These change the size of the graph:
    • \( af(x) \) stretches or compresses the graph vertically by a factor of \( a \). If \( a > 1 \), the graph is stretched; if \( 0 < a < 1 \), the graph is compressed.
    • \( f(bx) \) stretches or compresses the graph horizontally by a factor of \( \frac{1}{b} \). If \( b > 1 \), the graph is compressed (squeezed) horizontally; if \( 0 < b < 1 \), it is stretched.

Example:

Given \( f(x) = x^2 \):

  • Vertical translation: \( f(x) + 3 \) shifts the graph up by 3 units along the y-axis.
  • Horizontal translation: \( f(x - 2) \) shifts the graph right by 2 units along the x-axis.
  • Reflection in the x-axis: \( -f(x) = -x^2 \) flips the graph upside down.
  • Reflection in the y-axis: \( f(-x) = (-x)^2 = x^2 \) results in no change for this function, as it's symmetric about the y-axis.
  • Vertical stretch: \( 2f(x) = 2x^2 \) stretches the graph vertically by a factor of 2.
  • Horizontal compression: \( f(2x) = (2x)^2 = 4x^2 \) compresses the graph horizontally by a factor of \( \frac{1}{2} \).


Expressing Translations in Vector Form

Translations of functions can be expressed using vectors. A translation moves a graph horizontally, vertically, or both without altering its shape. We can describe translations in vector form as:

\[ \begin{pmatrix} a \\ b \end{pmatrix} \]

In this vector, \( a \) represents the horizontal shift, and \( b \) represents the vertical shift:

  • If \( a > 0 \), the graph shifts to the right by \( a \) units. If \( a < 0 \), the graph shifts to the left by \( a \) units.
  • If \( b > 0 \), the graph shifts upward by \( b \) units. If \( b < 0 \), the graph shifts downward by \( b \) units.

For example, the translation of the function \( f(x) = x^2 \) by 2 units to the left and 3 units up can be represented as:

\[ \begin{pmatrix} -2 \\ 3 \end{pmatrix} \]

This means the new function is \( f(x) = (x + 2)^2 + 3 \).

Partial Fractions

Partial fraction decomposition is a method used to express a rational function as a sum of simpler fractions. This technique is particularly useful in simplifying algebraic expressions and making them more manageable.

For example: A rational function like \( f(x) = \frac{2x + 3}{(x-1)(x+2)} \) can be expressed as the sum of two simpler fractions, making it easier to manipulate algebraically.

In this case, it can be written as: \[ f(x) = \frac{A}{x - 1} + \frac{B}{x + 2} \] where \( A \) and \( B \) are constants to be determined.


Step-by-Step Guide to Decomposing Rational Functions

Follow these steps to break down a rational function into partial fractions:

  1. Factorise the denominator: Factorise the denominator into linear terms.
    For example, for the rational function \[ \frac{2x + 1}{x^2 - 3x + 2} \] factor the denominator \[ x^2 - 3x + 2 \] into \[ (x - 1)(x - 2) \]
  2. Set up the partial fractions: For each factor in the denominator, write a fraction with an unknown numerator.
    Example: For \[ f(x) = \frac{2x + 1}{(x - 1)(x - 2)} \] we decompose it as: \[ f(x) = \frac{A}{x - 1} + \frac{B}{x - 2} \] where \( A \) and \( B \) are constants we need to solve for.
  3. Multiply through by the denominator: To eliminate the fractions, multiply both sides of the equation by the common denominator. This allows you to work with simpler equations without fractions.
    Start with the decomposed form: \[ \frac{2x + 1}{(x - 1)(x - 2)} = \frac{A}{x - 1} + \frac{B}{x - 2} \]
    Since all terms now have the common denominator \((x - 1)(x - 2)\), you can combine the fractions: \[ \frac{A(x - 2) + B(x - 1)}{(x - 1)(x - 2)} \]
    The equation with the common denominator is: \[ \frac{2x + 1}{(x - 1)(x - 2)} = \frac{A(x - 2) + B(x - 1)}{(x - 1)(x - 2)} \]
    Because the denominators on both sides of the equation are the same, you can ignore the denominators and focus only on the numerators: \[ 2x + 1 = A(x - 2) + B(x - 1) \]
  4. Expand and group terms: Expand the right-hand side and group terms by powers of \( x \).
    In this example, expand: \[ A(x - 2) + B(x - 1) \] which becomes \[ Ax - 2A + Bx - B \]
  5. Set up equations for coefficients: Compare the coefficients of the corresponding powers of \( x \) from both sides of the equation to form simultaneous equations.
    For example, if the left-hand side is \[ (2x + 1) \] compare the coefficients of \( x \) and the constant terms from both sides:
    For \( x \): \[ A + B = 2 \]
    For constants: \[ -2A - B = 1 \]
  6. Solve for constants: Solve the simultaneous equations to find the values of \( A \) and \( B \).
    In this example, solving \[ A + B = 2 \] and \[ -2A - B = 1 \] gives \[ A = 1 \] and \[ B = 1 \]
  7. Write the final decomposition: Substitute the constants back into the partial fractions.
    So, \[ \frac{2x + 1}{(x - 1)(x - 2)} = \frac{1}{x - 1} + \frac{1}{x - 2} \]

Try this exam-style question for yourself!

Decompose the following rational function into partial fractions:

\( \frac{3x}{(x + 1)(x - 2)} \)

A. \( \frac{2}{x - 2} + \frac{1}{x + 1} \)
B. \( \frac{1}{x - 2} + \frac{2}{x + 1} \)
C. \( \frac{3}{x - 2} - \frac{3}{x + 1} \)
D. \( \frac{2}{x + 1} - \frac{1}{x - 2} \)

Solution:

  1. Identify the function to decompose: \( \frac{3x}{(x + 1)(x - 2)} \).
  2. Set up the partial fractions: Decompose the function into: \[ \frac{3x}{(x + 1)(x - 2)} = \frac{A}{x - 2} + \frac{B}{x + 1} \]
  3. Multiply through by the denominator \((x + 1)(x - 2)\) to eliminate the fractions: \[ 3x = A(x + 1) + B(x - 2) \]
  4. Expand and simplify: \[ A(x + 1) + B(x - 2) = Ax + A + Bx - 2B \] Which simplifies to: \[ (A + B)x + (A - 2B) \]
  5. Compare coefficients from both sides of the equation: \[ 3x = (A + B)x + (A - 2B) \] For the coefficients of \(x\): \[ A + B = 3 \] And for the constant terms: \[ A - 2B = 0 \]
  6. Solve the system of equations: \[ A + B = 3 \] \[ A - 2B = 0 \] Solving these equations gives \( A = 2 \) and \( B = 1 \).
  7. Write the final decomposition: \[ \frac{3x}{(x + 1)(x - 2)} = \frac{2}{x - 2} + \frac{1}{x + 1} \] The correct answer is A.

Another example: A rational function like \( f(x) = \frac{4x + 5}{(x - 1)^2(x + 2)} \) can be expressed as the sum of simpler fractions, making it easier to manipulate algebraically.

In this case, it can be written as: \[ f(x) = \frac{A}{x - 1} + \frac{B}{(x - 1)^2} + \frac{C}{x + 2} \] where \( A \), \( B \), and \( C \) are constants to be determined.


Step-by-Step Guide to Decomposing Rational Functions with Repeated Linear Factors

Follow these steps to break down a rational function into partial fractions when the denominator includes repeated linear factors:

  1. Factorise the denominator: Factorise the denominator into linear terms, including any repeated factors.
    For example, for the rational function \[ \frac{4x + 5}{(x - 1)^2(x + 2)} \] the denominator is already factored into \[ (x - 1)^2(x + 2) \]
  2. Set up the partial fractions: For each factor, including repeated factors, write a fraction with an unknown numerator.
    Example: For \[ f(x) = \frac{4x + 5}{(x - 1)^2(x + 2)} \] decompose it as: \[ f(x) = \frac{A}{x - 1} + \frac{B}{(x - 1)^2} + \frac{C}{x + 2} \] where \( A \), \( B \), and \( C \) are constants we need to solve for.
    Note: When you have a repeated factor like \( (x - 1)^2 \) in the denominator, you need to include separate terms for each power of that factor. So, for \( (x - 1)^2 \), you use \(\frac{A}{x - 1}\) and \(\frac{B}{(x - 1)^2}\) to account for both parts. This way, you capture the complete effect of the repeated factor in the decomposition.
  3. Multiply through by the denominator: To eliminate the fractions, multiply both sides of the equation by the common denominator. This allows you to work with simpler equations without fractions.
    Start with the decomposed form: \[ \frac{4x + 5}{(x - 1)^2(x + 2)} = \frac{A}{x - 1} + \frac{B}{(x - 1)^2} + \frac{C}{x + 2} \]
    By adding the fractions on the right side, you get: \[ \frac{A(x - 1)^2(x + 2) + B(x-1)(x + 2) + C(x - 1)(x - 1)^2}{(x - 1)(x - 1)^2(x + 2)} \]
    A factor of \((x-1)\) cancels out from every term on the right hand side, leaving you with: \[ \frac{4x + 5}{(x - 1)^2(x + 2)} = \frac{A(x - 1)(x + 2) + B(x + 2) + C(x - 1)^2}{(x - 1)^2(x + 2)} \]
    Since the denominators on both sides are the same, you can focus only on the numerators: \[ 4x + 5 = A(x - 1)(x + 2) + B(x + 2) + C(x - 1)^2 \]
  4. Expand and group terms: Expand the right-hand side and group terms by powers of \( x \).
    In this example, expand: \[ A(x - 1)(x + 2) + B(x + 2) + C(x - 1)^2 \] which becomes \[ A(x^2 + x - 2) + B(x + 2) + C(x^2 - 2x + 1) \]
    This simplifies to \[ (A + C)x^2 + (A - 2C + B)x + (-2A + 2B + C) \]
  5. Set up equations for coefficients: Compare the coefficients of the corresponding powers of \( x \) from both sides of the equation to form simultaneous equations.
    For example, if the left-hand side is \[ 4x + 5 \] compare the coefficients of \( x^2 \), \( x \), and the constant terms from both sides:
    For \( x^2 \): \[ A + C = 0 \]
    For \( x \): \[ A - 2C + B = 4 \]
    For constants: \[ -2A + 2B + C = 5 \]
  6. Solve for constants: Solve the simultaneous equations to find the values of \( A \), \( B \), and \( C \).
    In this example, solving \[ A + C = 0 \] \[ A - 2C + B = 4 \] and \[ -2A + 2B + C = 5 \] gives \[ A = 1 \] \[ B = 5 \] \[ C = -1 \]
  7. Write the final decomposition: Substitute the constants back into the partial fractions.
    So, \[ \frac{4x + 5}{(x - 1)^2(x + 2)} = \frac{1}{x - 1} + \frac{5}{(x - 1)^2} - \frac{1}{x + 2} \]

Try this exam-style question for yourself!

Decompose the following rational function into partial fractions:

\[ \frac{4x + 13}{(x - 2)^2 (x + 1)} \]

A. \(\frac{1}{x + 1} - \frac{1}{x - 2} + \frac{7}{(x - 2)^2}\)
B. \(\frac{2}{x + 1} - \frac{3}{x - 2} + \frac{4}{(x - 2)^2}\)
C. \(\frac{3}{x + 1} - \frac{5}{x - 2} + \frac{2}{(x - 2)^2}\)
D. \(\frac{4}{x + 1} - \frac{6}{x - 2} + \frac{5}{(x - 2)^2}\)

Solution:

  1. Start with the partial fraction decomposition setup:

    2. \[ \frac{4x + 13}{(x - 2)^2 (x + 1)} = \frac{A}{x - 2} + \frac{B}{(x - 2)^2} + \frac{C}{x + 1} \]

  2. Multiply through by the common denominator \((x - 2)^2 (x + 1)\) to clear the fractions:

    3. \[ 4x + 13 = A(x - 2)(x + 1) + B(x + 1) + C(x - 2)^2 \]

  3. Expand and simplify:

    4. Expand \( A(x - 2)(x + 1) \): \[ A(x^2 - x - 2) \] Expand \( C(x - 2)^2 \): \[ C(x^2 - 4x + 4) \] Combine everything: \[ 4x + 13 = A(x^2 - x - 2) + B(x + 1) + C(x^2 - 4x + 4) \] \[ = (A + C)x^2 + (-A - 4C + B)x + (-2A + B + 4C) \]

  4. Equate coefficients with \(4x + 13\):

    5. Comparing coefficients: \[ A + C = 0 \] \[ -A - 4C + B = 4 \] \[ -2A + B + 4C = 13 \]

  5. Solve the system of equations:

    6. From \(A + C = 0\), we have \(C = -A\). Substitute \(C = -A\) into the other equations: \[ -A - 4(-A) + B = 4 \Rightarrow 3A + B = 4 \] \[ -2A + B + 4(-A) = 13 \Rightarrow -6A + B = 13 \] Solving these gives: \[ A = -1, \quad B = 7, \quad C = 1 \]

  6. Write the final partial fractions decomposition:

    7. \[ \frac{4x + 13}{(x - 2)^2 (x + 1)} = \frac{1}{x + 1} - \frac{1}{x - 2} + \frac{7}{(x - 2)^2} \]

Use of Functions in Modelling

Functions are widely used in various fields to create models that represent real-world situations. These models can help predict outcomes, understand relationships between variables, and make informed decisions. Below are examples of different types of models:

1. Linear Models

Linear models describe a relationship between two variables where the change in one variable is proportional to the change in another. These models are often used to predict values based on a linear relationship.

Example: Suppose the cost of hiring a car is £30 per day plus a one-time fee of £50. The total cost \( C \) for \( d \) days can be represented by the linear function:

\[ C(d) = 30d + 50 \]

Exam-Style Question:

Part (a): Create a linear model for the total cost of hiring a car if the cost per day is £25 and the one-time fee is £40.

Part (b): Calculate the total cost for 5 days using the model you created.

Part (c): Discuss a limitation of this model in a real-world context.

Solution:

  1. Part (a): To create the linear model, use the formula for total cost \( C \) where \( d \) is the number of days:

    \[ C(d) = \text{(Cost per day)} \times d + \text{(One-time fee)} \]

    Given that the cost per day is £25 and the one-time fee is £40, the model is:

    \[ C(d) = 25d + 40 \]

  2. Part (b): To find the total cost for 5 days, substitute \( d = 5 \) into the model:

    \[ C(5) = 25 \times 5 + 40 \]

    Calculate:

    \[ C(5) = 125 + 40 = £165 \]

  3. Part (c): A limitation of this model is that it does not account for possible discounts for long-term rentals or additional charges for extra services. It assumes a constant rate of cost per day without considering real-world variables such as promotions or additional fees.

2. Polynomial Models

Polynomial models involve functions where the relationship between variables is represented by a polynomial. These models can describe more complex relationships than linear models and are useful for data that show curves or varying rates of change.

Example: A company wants to model the profit \( P \) from selling \( x \) units of a product. The profit function can be represented as:

\[ P(x) = -2x^2 + 30x - 50 \]

Exam-Style Question:

Part (a): Calculate the profit when 10 units are sold using the model you created.

Part (b): Discuss a limitation of this polynomial model in a real-world context.

Solution:

  • Part (a): To find the profit when 10 units are sold, substitute \( x = 10 \) into the model:

    \[ P(10) = -10^2 + 20 \times 10 - 100 \]

    Calculate:

    \[ P(10) = -100 + 200 - 100 = £0 \]

  • Part (b): A limitation of this model is that it assumes profit decreases quadratically after a certain number of units. In reality, factors such as market saturation or changes in production costs might affect the profit differently, and the model may not accurately represent these real-world complexities.

Understanding Modulus Functions

In A-Level mathematics, the modulus function, denoted as \( |f(x)| \), plays a crucial role in solving equations and inequalities where the output must be non-negative. The modulus function can be understood as follows:

For a function \( f(x) \), the modulus function \( |f(x)| \) is defined as:

\[ |f(x)| = \begin{cases} f(x) & \text{if } f(x) \geq 0 \\ -f(x) & \text{if } f(x) < 0 \end{cases} \]

This means that \( |f(x)| \) outputs the absolute value of \( f(x) \). If \( f(x) \) is positive or zero, \( |f(x)| \) is just \( f(x) \). If \( f(x) \) is negative, \( |f(x)| \) is the positive version of \( f(x) \). This is particularly useful for modeling scenarios where only the magnitude of a quantity matters, regardless of its direction or sign.

Example: Consider the modulus function \( |x - 4| \). This function will produce the same output as \( x - 4 \) when \( x \geq 4 \), but will produce \( 4 - x \) when \( x < 4 \). Therefore:

  • For \( x \geq 4 \), \( |x - 4| = x - 4 \)
  • For \( x < 4 \), \( |x - 4| = 4 - x \)

This understanding helps in solving equations and inequalities involving modulus functions by breaking them into simpler linear equations or inequalities based on the definition of the modulus function.

Modulus functions can be used to solve equations where the expression inside the modulus could be positive or negative. To solve such equations, you need to consider both cases separately.

Solving Modulus Equations

To solve an equation involving a modulus, follow these steps:

  1. Isolate the modulus on one side of the equation.
  2. Set up two separate equations based on the definition of the modulus:
    • One equation where the expression inside the modulus is equal to the right-hand side of the equation.
    • Another equation where the expression inside the modulus is equal to the negative of the right-hand side of the equation.
  3. Solve each of the equations separately.
  4. Check all solutions in the original equation to ensure they are valid.

Example: Solve the equation \( |x - 3| = 5 \).

Solution:

  1. Isolate the modulus: \( |x - 3| = 5 \).
  2. Set up the two cases:
    • Case 1: \( x - 3 = 5 \)
    • Case 2: \( x - 3 = -5 \)
  3. Solve each case:
    • For Case 1: \( x - 3 = 5 \) ⟹ \( x = 8 \)
    • For Case 2: \( x - 3 = -5 \) ⟹ \( x = -2 \)
  4. The solutions are \( x = 8 \) and \( x = -2 \). Check both solutions in the original equation to confirm they are valid.

Try this exam-style question for yourself!

Solve the modulus inequality:

\[ |x - 4| > 2 \]

A. \( x < 2 \text{ or } x > 6 \)
B. \( x < 6 \text{ or } x > 2 \)
C. \( x < 2 \text{ or } x > 8 \)
D. \( x < 6 \text{ or } x > 4 \)

Solution:

  1. Start with the modulus inequality: \( |x - 4| > 2 \).
  2. Break it into two separate inequalities:
    • Case 1: \( x - 4 > 2 \)
    • Case 2: \( -(x - 4) > 2 \), which simplifies to \( x - 4 < -2 \)
    • Note:Remember, when solving inequalities, if you multiply or divide by a negative number, you must flip the inequality sign.
  3. Solve each case:
    • For Case 1: \( x - 4 > 2 \)
      • Add 4 to both sides: \( x > 6 \)
    • For Case 2: \( x - 4 < -2 \)
      • Add 4 to both sides: \( x < 2 \)
  4. Combine the results from both cases: \( x < 2 \text{ or } x > 6 \).
  5. So, the solution is \( x < 2 \text{ or } x > 6 \). Check the solution by substituting values from both ranges back into the original inequality to confirm they satisfy it.

Solving Modulus Inequalities

To solve inequalities involving modulus functions, follow similar steps as for equations, but focus on the range of values that satisfy the inequality. For example, to solve \( |x - 3| < 5 \), you would set up the inequality as:

\[-5 < x - 3 < 5\]

Then solve the compound inequality:

\[-5 < x - 3 < 5\]

Add 3 to all parts:

\[-2 < x < 8\]

So, the solution is \( -2 < x < 8 \).

Solving Modulus Equations by Squaring Both Sides

When solving equations that involve modulus functions, you might need to square both sides to get rid of the modulus. Here’s how to do it:

For example, consider the equation \( |x - 2| = x + 1 \). Follow these steps:

1. **Square Both Sides**: To eliminate the absolute value, square both sides of the equation:

\[(|x - 2|)^2 = (x + 1)^2\]

This simplifies to:

\[(x - 2)^2 = (x + 1)^2\]

2. **Expand and Simplify**: Next, expand both sides:

\[x^2 - 4x + 4 = x^2 + 2x + 1\]

Subtract \(x^2\) from both sides and simplify:

\[-4x + 4 = 2x + 1\]

Combine like terms:

\[-6x = -3\]

So:

\[x = \frac{1}{2}\]

3. **Check Your Solution**: Always substitute your solution back into the original equation to make sure it works:

For \( x = \frac{1}{2} \), check:

\[|\frac{1}{2} - 2| = \frac{1}{2} + 1\]

\[|\frac{-3}{2}| = \frac{3}{2}\]

This is true, so \( x = \frac{1}{2} \) is a valid solution.

**Why Square Both Sides?**: Squaring both sides helps to remove the modulus function, which simplifies the equation. However, squaring can sometimes introduce extra solutions, so it’s important to check each solution in the original equation.
