Inequalities


Solving Linear Inequalities

Linear inequalities are similar to linear equations but involve an inequality sign instead of an equal sign. To solve a linear inequality, follow these steps:

  • Isolate the variable on one side of the inequality.
  • Perform the same operations on both sides, just as you would with equations.
  • Remember to reverse the inequality sign if you multiply or divide both sides by a negative number.

Example: Solve the inequality:

\[ -3x + 4 < 1 \]

Subtract 4 from both sides:

\[ -3x < -3 \]

Divide both sides by -3 (note the sign flip):

\[ x > 1 \]

Important Point: When dividing or multiplying by a negative number, flip the direction of the inequality sign.

To represent the inequality \( x > 1 \) graphically, draw a vertical dotted line at \( x = 1 \). The line is dotted because the inequality does not include an equal sign (i.e., \( x \neq 1 \)). This indicates that values strictly less than 1 are solutions, but 1 itself is not included. The shaded region is the region \(x>1\).

Solve the inequality:

\[ 3x - 5 \leq 7 \]

A. \( x \leq 4 \)
B. \( x \geq 4 \)
C. \( x \leq 6 \)
D. \( x \geq 6 \)

Solution:

  1. Add 5 to both sides: \( 3x - 5 + 5 \leq 7 + 5 \) which simplifies to \( 3x \leq 12 \).
  2. Divide both sides by 3: \( \frac{3x}{3} \leq \frac{12}{3} \) which simplifies to \( x \leq 4 \).
  3. Therefore, the correct answer is \( x \leq 4 \).

Solving Quadratic Inequalities

Quadratic inequalities involve expressions of the form \( ax^2 + bx + c \), where \( a \), \( b \), and \( c \) are constants. To solve these using graphical methods:

  • Graph the Quadratic Function: Plot the function \( f(x) = ax^2 + bx + c \) to visualise the shape of the curve and identify the roots.
  • Find the Roots: Solve the quadratic equation \( ax^2 + bx + c = 0 \) to find the x-values where the graph intersects the x-axis. These points are crucial as they divide the x-axis into intervals.
  • Analyse the Curve: Based on the shape of the curve (whether it is U-shaped or N-shaped) and the points where it intersects the x-axis, determine where the inequality holds true.

Critical Values: These are the x-values where the quadratic expression equals zero. They are used to divide the number line into intervals to test the inequality.

Example 1: Solve the inequality:

\[ x^2 - 4x - 5 < 0 \]

First, solve the quadratic equation:

\[ x^2 - 4x - 5 = 0 \]

Factorising gives:

\[ (x - 5)(x + 1) = 0 \]

The roots are \( x = 5 \) and \( x = -1 \).

Graph the quadratic function \( f(x) = x^2 - 4x - 5 \). The graph will intersect the x-axis at \( x = 5 \) and \( x = -1 \). The curve is U-shaped (since the coefficient of \( x^2 \) is positive).

To find where \( f(x) < 0 \), observe the shape of the curve and the intervals created by the roots:

In the graph of \( x^2 - 4x - 5 = 0 \), we focus on the parts of the curve where the function is less than zero, as required by the inequality (in the question). This means we are interested in the portions of the curve that lie below the x-axis. For this particular function, these regions occur between the roots, so the solution to the inequality \( x^2 - 4x - 5 < 0 \) is \[ -1 < x < 5 \].

Example 2: Solve the inequality:

\[ x^2 - 4x + 3 \geq 0 \]

First, solve the quadratic equation:

\[ x^2 - 4x + 3 = 0 \]

Factorising gives:

\[ (x - 3)(x - 1) = 0 \]

The roots are \( x = 3 \) and \( x = 1 \).

Graph the quadratic function \( f(x) = x^2 - 4x + 3 \). The graph will intersect the x-axis at \( x = 3 \) and \( x = 1 \). The curve is U-shaped (since the coefficient of \( x^2 \) is positive).

To find where \( f(x) \geq 0 \), observe the shape of the curve and the intervals created by the roots:

In the graph of \( x^2 - 4x + 3 = 0 \), we focus on the parts of the curve where the function is greater than or equal to zero, as required by the inequality. This means we are interested in the portions of the curve that lie above or on the x-axis. For this particular function, these regions occur outside the interval between the roots, so the solution to the inequality \( x^2 - 4x + 3 \geq 0 \) is

\( x \leq 1 \) or \( x \geq 3 \).

Using Desmos to Graph Quadratic Inequalities:

  • Enter the quadratic function into Desmos.
  • Identify the roots where the graph intersects the x-axis.
  • Observe the regions above or below the x-axis to determine where the inequality is satisfied.

Set Notation

In A-Level mathematics, set notation is used to describe the solution sets of inequalities. The notation includes the following symbols:

  • Union (\( \cup \)): Represents the union of two sets, combining all elements from both sets. For example, if \( A \) and \( B \) are sets, then \( A \cup B \) is the set of elements that are in \( A \), in \( B \), or in both.
  • Intersection (\( \cap \)): Represents the intersection of two sets, including only the elements that are in both sets. For example, \( A \cap B \) is the set of elements that are in both \( A \) and \( B \).
  • Set Builder Notation: Used to describe sets by a property that its members must satisfy. For instance, the set of all \( x \) such that \( x \) is greater than 3 can be written as \( \{ x : x > 3 \} \).
  • Example: For the solution to the inequality \( x^2 - 4x + 3 \geq 0 \), we use set notation to describe the solution set as \( x \leq 1 \) or \( x \geq 3 \). This can be written in set builder notation as:

\[\{ x : x \leq 1 \} \cup \{ x : x \geq 3 \}\]

Additional Example: For the solution to the inequality \( x^2 - 2x - 3 < 0 \), we use set notation to describe the solution set as \( -1 < x < 3 \). This can be written in set notation as:

\[\{ x : -1 < x < 3 \}\]

Let's try a question that relates both the quadratic inequality stuff you learnt and the set notation stuff

Solve the inequality:

\[ x^2 - 2x - 3 < 0 \]

A. \( \{ x : x < -1 \} \cup \{ x : x > 3 \} \)
B. \( \{ x : -1 < x < 3 \} \)
C. \( \{ x : x \leq -1 \} \cup \{ x \geq 3 \} \)
D. \( \{ x : x \leq 3 \} \cup \{ x \geq -1 \} \)

Solution:

  1. Solve the quadratic equation \( x^2 - 2x - 3 = 0 \). Factoring gives \( (x - 3)(x + 1) = 0 \), with roots \( x = 3 \) and \( x = -1 \).
  2. Graph the quadratic function \( f(x) = x^2 - 2x - 3 \). The graph intersects the x-axis at \( x = 3 \) and \( x = -1 \). The curve is U-shaped (since the coefficient of \( x^2 \) is positive).
  3. To find where \( f(x) < 0 \), observe the regions between the roots where the curve is below the x-axis.
  4. The solution to the inequality \( x^2 - 2x - 3 < 0 \) is the interval between the roots, so it can be written in set notation as \( \{ x : -1 < x < 3 \} \).

Topic Review: Inequalities

In this topic, you have learned the following key concepts:

  • Linear Inequalities: Linear inequalities involve expressions of the form \( ax + b \), where \( a \) and \( b \) are constants. To solve a linear inequality, isolate the variable and perform the same operations on both sides, remembering to reverse the inequality sign if you multiply or divide by a negative number.
  • Quadratic Inequalities: Quadratic inequalities involve expressions of the form \( ax^2 + bx + c \), where \( a \), \( b \), and \( c \) are constants. To solve these, graph the quadratic function to determine where it is above or below the x-axis, or use algebraic methods to find the intervals where the inequality holds true.
  • Graphical Solutions: For quadratic inequalities, graph the function and use the shape of the curve to determine the regions where the inequality is satisfied. Identify the roots and analyze the intervals created by these critical values.
  • Critical Values: These are the x-values where the expression equals zero. They divide the number line into intervals which can be tested to find where the inequality is satisfied.
  • Set Notation: Set notation is used to express the solution sets of inequalities. For example, \( x^2 - 4x + 3 \geq 0 \) can be written in set notation as \( \{ x : x \leq 1 \} \cup \{ x : x \geq 3 \} \).
  • Using Desmos: Desmos can be used to graph inequalities and visualize the solution sets. Enter the function, identify the roots, and observe the regions above or below the x-axis to solve the inequality.
